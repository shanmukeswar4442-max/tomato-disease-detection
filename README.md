# Tomato Disease Detection System

AI powered tomato disease detection for farmers.

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: FastAPI, Python
- ML: YOLOv8 (97.8% accuracy)
- Database: PostgreSQL
- AI: Google Gemini API
- Deployment: AWS

## Features
- Detect 11 tomato diseases
- AI treatment suggestions
- Prediction history
- User authentication
- Dashboard analytics

## Setup
1. Clone repository
2. Setup backend: pip install -r requirements.txt
3. Setup frontend: npm install
4. Configure .env file
5. Run backend: uvicorn main:app --reload
6. Run frontend: npm run dev
